1//! SNES DMA + HDMA controllers.
2//!
3//! P1.2 scope: synchronous DMA only (HDMA in a later phase). Eight
4//! channels, all 8 transfer modes, A→B and B→A directions, with the
5//! A-bus increment / decrement / fixed behaviours.
6//!
7//! Bus abstraction: the DMA logic is decoupled from `luna-core`'s
8//! `SnesBus` via the [`DmaBus`] trait, which exposes the minimum
9//! primitives DMA needs (`read_a` / `write_a` on the 24-bit CPU bus,
10//! `read_b` / `write_b` on the PPU's 8-bit `$2100 + offset` bus). The
11//! production wiring lives in `luna-core`; this crate is testable in
12//! isolation with mock buses.
13//!
14//! See `ARCHITECTURE.md` §6.4.
15
