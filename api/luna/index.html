write_dma_trace_csv 🔒
Write DMA→VRAM transfer-time trace events as CSV. Columns: seq,frame,line,blank,force_blank,src,vram_word,reg,value — blank is the V-blank flag, force_blank INIDISP ($2100) bit 7 at the write (safe iff blank||force_blank), src the 24-bit A-bus source (bank:offset), vram_word the VMADD word the byte landed at, reg the B-bus port ($2118 low / $2119 high), value the transferred byte.
