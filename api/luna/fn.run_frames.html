luna frames — capture count exactly-consecutive PPU frames as PNGs via the same luna-api render path the GUI uses, tagging each with its frame number and forced-blank flag. Lets us reproduce the temporal artefacts (flicker / page-flip desync) that a single state --screenshot is structurally blind to.
