pub fn apply_brightness(rgb: [u8; 3], brightness: u8) -> [u8; 3]
Expand description

Apply SNES master brightness ($2100 bits 0-3, 0..15) to an already-converted RGB888 triple.

Per ares color.cpp, L = (1 + l)/16 * (l ? 1.0 : 0.25): brightness 1..15 scale linearly as in * (brightness + 1) / 16 (full at 15), and brightness 0 is an extra ÷4 darker — i.e. in / 64, not pure black. ares notes this is hardware-measured (“luma=0 is not 100% black; but much darker than linear scaling”); Mesen2 models it as pure black (in * b / 15), but ares is the hardware reference here.
