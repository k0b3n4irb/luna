Module event_viewer

Faithful Mesen2 SNES Event Viewer data layer (categories, colors, register names, config). The GUI Event Viewer panel consumes this via the accessors on Emulator. Faithful port of Mesen2’s SNES Event Viewer data layer — categorisation, default colors, register names, and config.

Everything here is ported verbatim from Mesen2 master (the watchword is fidelity); see docs/event_viewer_reference.md for the full study. Sources:

  • categorisation: Core/SNES/Debugger/SnesEventManager.cpp GetEventConfig
  • default colors: UI/Config/Debugger/SnesEventViewerConfig.cs
  • register names: UI/Debugger/Labels/DefaultLabelHelper.cs SetSnesDefaultLabels
