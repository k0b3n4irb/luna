Expand description

SNES memory map, cartridge mappers and the Bus trait.

The Bus trait exposes Bus::io_cycle — the key primitive that makes mid-instruction PPU/HDMA catch-up possible (and thus correct Mario Kart, F-Zero, and every other HDMA-heavy SNES game).

§Crate layout

  • types: time / address aliases (MCycles, address helpers).
  • speed: SNES memory access speed lookup (FAST / SLOW / XSLOW).
  • bus: Bus and BusDevice traits.
  • mapper: Mapper trait for cartridge mappings.
  • lorom: lorom::LoRomMapper — Mode 20 cartridge mapping.
  • [mod@testing]: [testing::RamBus] — a flat-RAM Bus for unit tests in downstream crates (gated behind the test-utils feature or #[cfg(test)]).

See ARCHITECTURE.md §5.
