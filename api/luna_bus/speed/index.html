Module speed 

Source
Expand description

SNES memory access speed lookup.

Each region of the SNES memory map is accessed at one of three speeds. The cost is in master clock cycles per byte access, and is paid via crate::Bus::io_cycle at every read or write.

References: https://problemkaputt.de/fullsnes.htm §“SNES Memory Map” and §“SNES Mem Speed (FastROM / SlowROM)”.
