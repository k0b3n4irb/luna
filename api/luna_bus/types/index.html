rom_mirror
Fold a linear ROM address into a size-byte image, mirroring exactly as SNES hardware does. Faithful port of ares’ Bus::mirror (sfc/memory/inline.hpp). For a power-of-two size this is just address % size; for a non-power-of-two cart (e.g. 1.5 MB) the region past the end mirrors the trailing largest-power-of-two chunk rather than returning open bus. size == 0 yields 0 (caller treats an empty ROM as unmapped).
